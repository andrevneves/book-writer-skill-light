# Scene Card Template

**Purpose:** To plan scenes at the atomic level before drafting. A scene is the smallest unit of fiction — every scene must have conflict; a scene without conflict is summary, not a scene.

Use one card per scene. Store scene cards inside the relevant chapter outline file, or in a separate `Scene_Cards/` directory.

## Integration Notes for the AI
*Read scene cards before drafting their chapter. Do NOT add new scene goals or outcomes not specified unless the outline explicitly marks a beat as flexible. Each scene should exit differently than it entered — POV character's emotional state and factual situation must shift. Use Character State at Start to enter the scene and Character State Now / Exit State to update the next scene's previous context.*
